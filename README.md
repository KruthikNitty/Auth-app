#Auth  App
its an secure authentication app 
built with React and Node.js

Frontend:React(Vite)
Backend:Node.js+Express
Authentication:JWT(JSON Web Token)

##Backend Setup

cd backend
npm install
node server.js

server runs on :
http:/localhost:5000


#Frontend Sewtup

cd frontend
npm install
npm run dev

server runs on :
http:/localhost:5173
